# useFileUploader

A hook used is used to connect [FileUploadService\<Response\>](../types/FileUploadService) and [FileUploaderObservers\<Response\>](../types/FileUploaderObservers)
which contains [FileUploadManager\<Response\>](../types/FileUploadManager). Particularly useful when using Low Level Components to 
build custom High Level Components.

Returns: [FileUploader\<Response\>](../types/FileUploader)
