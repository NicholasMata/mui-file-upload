# FileDropzoneBody

This component uses [useFileDropzoneContext](../../hooks/useFileDropzoneContext) to provide the default body for 
[MultiFileUpload](../basics/MultiFileUpload), it can also be used for [SingleFileUpload](../basics/SingleFileUpload). 

:::note
You can use this component to customize labels used in [MultiFilepload](../basics/MultiFileUpload) or [SingleFileUpload](../basics/SingleFileUpload).
:::

## Usage
<CustomFileDropzoneBodyUsage />

<br/>
```jsx
import { useXMLHttpService, MultiFileUpload, FileDropzoneBody } from 'mui-file-upload';

export const Component = () => {
  const uploadService = useXMLHttpService(...);
  return <MultiFileUpload
    uploadService={uploadService}
    body={
      <FileDropzoneBody
        title='Custom Title'
        dropTitle='Custom Drop Title'
        disabledTitle='Custom Disabled Title'
        dragRejectedTitle='Custom Rejected Title'
        fileOverloadTitle='Custom Too Many Files'
      />
    }
  />
}
```
